## CSS

- 글자 수가 많아서 기존 box를 넘어가거나 줄바꿈이 될 때 "글자가 초과합..."처럼 ...으로 만드는 3종 세트

  ```css
      white-space: nowrap;
      overflow: hidden;
      text-overflow: ellipsis;
  ```

  `white-space: nowrap;` : 넓이를 넘어가더라도 줄바꿈을 하지 않는다.

  `overflow: hidden;` : 넘어간 부분을 숨긴다.

  `text-overflow: ellipsis;` : overflow: hidden 된 부분을 ...으로 나타낸다.
